# ruppel — Тестовый HTML-опрос «Комфортно ли вашему ребенку в школе?»

**Фича:** Статическая HTML-страница с психологическим тестом для родителей (17 вопросов). Оценивает, насколько комфортно ребёнку в школе, выявляет признаки буллинга. Деплоится на GitHub Pages.

**Поток:**
1. Открыть `test1.html` в браузере.
2. Родитель отвечает на 17 вопросов (радио-кнопки: Никогда / Редко / Часто / Постоянно).
3. Все вопросы обязательны — при пропуске показывается ошибка.
4. После ответа на все вопросы — нажать «Узнать результат».
5. Подсчёт суммы баллов (0–3 за вопрос, макс. 51). Вывод цветного результата:
   - **0–17 баллов (зелёный):** всё хорошо.
   - **18–34 балла (жёлтый):** есть повод для беспокойства, рекомендации.
   - **35–51 балл (красный):** высокая вероятность буллинга, срочные рекомендации.
6. GitHub Actions (`static.yml`) автоматически деплоит на GitHub Pages при пуше в main.

**Что важно:**
- **Статический HTML.** Ни бэкенда, ни сохранения данных. Результат вычисляется на клиенте JS.
- **Файл сгенерирован из Google Docs / через Cocoa HTML Writer** (meta: Generator=Cocoa HTML Writer, форматирование через встроенные стили Times/Apple Color Emoji).
- **Единственный файл:** `test1.html` (821 строка) + README.
- **GitHub Pages:** воркфлоу `.github/workflows/static.yml` — checkout → configure-pages → upload-pages-artifact → deploy.
- **Нет отправки данных** — приватность, всё локально в браузере.
